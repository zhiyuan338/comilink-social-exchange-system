# ComiLink MVP 页面路由设计

## 公共页面

### `/login`

登录页。

功能：

- QQ 号登录
- 密码登录
- 登录后返回原访问页面

---

### `/u/[token]`

NFC 访问主页。

功能：

- 展示目标用户昵称
- 展示电子邮票
- 展示社交入口
- 未登录用户可查看
- 已登录用户自动尝试建立图鉴关系
- 自己访问自己时不建立关系
- token 不存在时显示 404

---

## 用户页面

### `/me`

用户资料页。

功能：

- 查看自己的 NFC 链接
- 修改昵称
- 上传电子邮票
- 管理社交入口
- 修改密码

---

### `/me/collections`

图鉴页。

功能：

- 查看已收集用户
- 按活动筛选
- 显示收集时间
- 显示对方昵称与电子邮票

---

## 后台页面

### `/admin`

后台首页。

功能：

- 展示用户数量
- 展示当前活动
- 快速进入用户管理与活动管理

---

### `/admin/users`

用户管理页。

功能：

- 查看用户列表
- 批量导入用户
- 查看用户 QQ、昵称、token
- 重置用户密码为 QQ 号

---

### `/admin/events`

活动管理页。

功能：

- 创建活动
- 查看活动列表
- 设置当前活动
- 查看当前活动

---

## 路由权限

| 路由 | 是否需要登录 | 是否需要管理员 |
|---|---:|---:|
| `/login` | 否 | 否 |
| `/u/[token]` | 否 | 否 |
| `/me` | 是 | 否 |
| `/me/collections` | 是 | 否 |
| `/admin` | 是 | 是 |
| `/admin/users` | 是 | 是 |
| `/admin/events` | 是 | 是 |
