# ComiLink MVP 第 0 步开发资料包

本资料包用于正式进入开发前的准备阶段，目标是让 AI 编程工具或开发者不需要反复猜测需求。

建议开发前先完整阅读以下文件：

1. `01-final-prd.md`：最终 PRD
2. `02-tech-stack.md`：技术栈确认
3. `03-database-schema.md`：数据库 Schema 设计
4. `04-routes.md`：页面路由设计
5. `05-api-spec.md`：API 输入输出规范
6. `06-acceptance-criteria.md`：MVP 验收标准
7. `07-env-vars.md`：环境变量清单
8. `08-development-order.md`：推荐开发顺序

核心开发目标：

> 触碰 NFC → 打开主页 → 自动建立双向图鉴关系 → 查看电子邮票与联系方式

本阶段不做即时聊天、动态社区、排行榜、任务系统、推送通知、复杂权限系统或用户注销功能。
